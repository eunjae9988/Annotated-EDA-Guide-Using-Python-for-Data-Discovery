# Annotated-EDA-Guide-Using-Python-for-Data-Discovery
This repo is my Coursera lab work for exploring datasets using Python.
I followed the course notebook to practice pandas, numpy, matplotlib, and seaborn.
It shows how to check dataset info, summarize data, group by date/month, and visualize results.
I didn’t need to write any original code; I just followed along and learned the steps.

Libraries Used
- **pandas** – for handling data like a spreadsheet. Helps clean, filter, and analyze tables.  
- **numpy** – for fast number and math operations, like sums, averages, and arrays.  
- **matplotlib** – for drawing basic graphs like bar charts and line plots.  
- **seaborn** – makes graphs look nicer, especially for statistics and distributions.  
Note: I completed this lab pretty quickly, so it’s mainly for practice and reference.

Dataset

The dataset used is 2018 lightning strike data from NOAA.

What’s inside:
EDA_python_lab.ipynb – Notebook with all the follow-along code
eda_using_basic_data_functions_in_python_dataset1.csv – Dataset file
Key skills practiced
Reading CSV files into Python
Using pandas methods: head(), info(), describe(), groupby()
Creating new columns and summarizing data
Plotting with matplotlib and seaborn
